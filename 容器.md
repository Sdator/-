
## 踩坑
+ 环境 
  + W10 平台
  + gitbash
+ 关键命令
  + $(cmd //c echo %CD%)
  
```bash
# Windows 平台下正确读取路径的方法
docker run --rm -v $(cmd //c echo %CD%):/tmp -it sdator/tools ls -al

# 错误例子 以下适合 linux 平台
docker run --rm -v `pwd`:/tmp -it sdator/tools ls -al
docker run --rm -v $(pwd):/tmp -it sdator/tools ls -al
docker run --rm -v "${pwd}":/tmp -it sdator/tools ls -al
```
