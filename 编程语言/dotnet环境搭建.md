

# Dotnet环境搭建



## 1.安装环境

### 手动下载SDK

https://dotnet.microsoft.com/zh-cn/download


### 使用 winget 下载【推荐】

```powershell
winget install --id Microsoft.dotnet
```

### 使用 scoop 下载

```powershell
scoop install dotnet-sdk
```

## 2.创建项目

```powershell
# 创建控制台程序
dotnet new console -o <项目名称>
```
