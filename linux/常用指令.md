>## curl
### 参数
+ -I 仅测试HTTP头
+ -m 10 最多查询10s
+ -o /dev/null 屏蔽原有输出信息
+ -s silent 模式，不输出任何东西
+ -w %{http_code} 控制额外输出
### 例子
```bash
# 获取状态码
curl -I -m 10 -o /dev/null -s -w %{http_code} www.baidu.com
curl -I -m 10 -o /dev/null -s -w %{http_code} youtube.com
```
